# Indian Political Speech Sentiment Analysis Using Transformer-Based NLP

## Project Overview

This project develops a multilingual NLP framework for sentiment analysis and rhetorical analysis of Indian political speeches from text data extracted from Youtube using transformer-based deep learning models.

The system combines:
- exploratory data analysis (EDA),
- n-gram based rhetorical pattern extraction,
- weakly supervised sentiment labeling,
- and multilingual transformer fine-tuning

to classify political speech sentiment into:
- Positive
- Neutral
- Negative

The project focuses on:
- political discourse analysis,
- contextual sentiment understanding,
- rhetorical pattern mining,
- and multilingual transformer-based NLP.

---

# Objectives

The primary objectives of this project are:

- Analyze Indian political speeches using NLP techniques.
- Perform sentence-level sentiment classification.
- Extract political rhetoric using bigram and trigram analysis.
- Study speaker-wise and year-wise discourse patterns.
- Fine-tune transformer models for political sentiment analysis.
- Explore future extensions toward stance detection.
